# Max Distance task

## Quick Start

### Prerequisites
* Python 3.9+
* `pip` or `pipx` (for the sake of cli tools isolation)
```shell


# ensure you have poetry
pipx install poetry  # or pip install poetry

cd interview-excersises
git checkout python_max_distance

poetry shell
poetry install

```
### Run the cli app
```shell
poetry shell # if not done yet
solve-md --help
```
